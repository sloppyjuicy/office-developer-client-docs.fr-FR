# <a name="pidlidemail2originalentryid-canonical-property"></a>Propriété canonique PidLidEmail2OriginalEntryId

  
  
**S’applique à** : Outlook 2013 | Outlook 2016 
  
Spécifie **l’EntryId** de l’objet qui correspond à la deuxième adresse e-mail. 
  
|Propriété|Valeur|
|:-----|:-----|
|Propriétés associées :  <br/> |dispidEmail2OriginalEntryID  <br/> |
|Jeu de propriétés :  <br/> |PSETID_Address  <br/> |
|ID long (LID) :  <br/> |0x00008095  <br/> |
|Type de données :  <br/> |PT_BINARY  <br/> |
|Domaine :  <br/> |Contact  <br/> |
   
## <a name="remarks"></a>Remarques

La valeur de cette propriété doit être un **EntryId** unique pour cette adresse électronique ou un **EntryId** d’objet carnet d’adresses valide.
