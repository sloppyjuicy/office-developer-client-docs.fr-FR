# <a name="pidlidhtml-canonical-property"></a>Propri t canonique PidLidHtml

  
  
**S’applique à** : Outlook 2013 | Outlook 2016 
  
Spécifie l’URL de la page Web professionnelle du contact.
  
|Propriété |Valeur |
|:-----|:-----|
|Propriétés associées :  <br/> |dispidHTML  <br/> |
|Jeu de propriétés :  <br/> |PSETID_Address  <br/> |
|ID long (LID) :  <br/> |0x0000802B  <br/> |
|Type de données :  <br/> |PT_UNICODE  <br/> |
|Domaine :  <br/> |Contact  <br/> |
   
## <a name="remarks"></a>Remarques

La valeur de cette propriété, si elle est présente, doit être identique à la valeur de la propriété **PR_BUSINESS_HOME_PAGE** ([PidTagBusinessHomePage](pidtagbusinesshomepage-canonical-property.md)).
