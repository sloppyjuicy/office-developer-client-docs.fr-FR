# <a name="pidtaginternetcontrol-canonical-property"></a>Propriété canonique PidTagInternetControl

  
  
**S’applique à** : Outlook 2013 | Outlook 2016 
  
Cartes à l’en-tête du contrôle MIME.
  
|Propriété |Valeur |
|:-----|:-----|
|Propriétés associées :  <br/> |PR_INTERNET_CONTROL, PR_INTERNET_CONTROL_A, PR_INTERNET_CONTROL_W  <br/> |
|Identificateur :  <br/> |0x1031  <br/> |
|Type de données :  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Domaine :  <br/> |MIME  <br/> |
