# <a name="pidtaghobbies-canonical-property"></a>PidTagHobbies Canonical, propriété

  
  
**S’applique à** : Outlook 2013 | Outlook 2016 
  
Contient les noms des passe-temps de l’utilisateur de messagerie.
  
|Propriété|Valeur|
|:-----|:-----|
|Propriétés associées :  <br/> |PR_HOBBIES, PR_HOBBIES_A, PR_HOBBIES_W  <br/> |
|Identificateur :  <br/> |0x3A43  <br/> |
|Type de données :  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Domaine :  <br/> |Utilisateur de messagerie MAPI  <br/> |
   
## <a name="remarks"></a>Remarques

Ces propriétés fournissent des informations d’identification et d’accès sur un utilisateur de messagerie et le contenu est défini par l’organisation de l’utilisateur.
