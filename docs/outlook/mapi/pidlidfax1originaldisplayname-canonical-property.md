# <a name="pidlidfax1originaldisplayname-canonical-property"></a>Propriété canonique PidLidFax1OriginalDisplayName

  
  
**S’applique à** : Outlook 2013 | Outlook 2016 
  
Spécifie le nom complet d’origine de l’adresse de télécopie professionnel du contact.
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |dispidFax1OriginalDisplayName  <br/> |
|Jeu de propriétés :  <br/> |PSETID_Address  <br/> |
|ID de type long (capot) :  <br/> |0x000080B4  <br/> |
|Type de données :  <br/> |PT_UNICODE  <br/> |
|Domaine :  <br/> |Contact  <br/> |
   
## <a name="remarks"></a>Remarques

Cette propriété, le cas échéant, doit avoir la même valeur que la propriété **PR_NORMALIZED_SUBJECT** ([PidTagNormalizedSubject](pidtagnormalizedsubject-canonical-property.md)).
