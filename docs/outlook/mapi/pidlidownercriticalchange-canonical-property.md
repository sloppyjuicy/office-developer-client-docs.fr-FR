# <a name="pidlidownercriticalchange-canonical-property"></a>Propriété canonique PidLidOwnerCriticalChange

  
  
**S’applique à** : Outlook 2013 | Outlook 2016 
  
Spécifie la date et l’heure auxquelles une demande de réunion a été envoyée par l’organisateur.
  
|Propriété|Valeur|
|:-----|:-----|
|Propriétés associées :  <br/> |LID_OWNER_CRITICAL_CHANGE  <br/> |
|Jeu de propriétés :  <br/> |PSETID_Meeting  <br/> |
|ID long (LID) :  <br/> |0x0000001A  <br/> |
|Type de données :  <br/> |PT_SYSTIME  <br/> |
|Domaine :  <br/> |Réunions  <br/> |
   
## <a name="remarks"></a>Remarques

La valeur doit être spécifiée en temps universel coordonné (UTC).
