# <a name="pidtagattachmentlinkid-canonical-property"></a>Propriété canonique PidTagAttachmentLinkId

  
  
**S’applique à** : Outlook 2013 | Outlook 2016 
  
Indique le type d’objet Message auquel cette pièce jointe est liée.
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |PR_ATTACHMENT_LINKID  <br/> |
|Identificateur :  <br/> |0x7FFA  <br/> |
|Type de données :  <br/> |PT_LONG  <br/> |
|Domaine :  <br/> |Pièce jointe de message  <br/> |
   
## <a name="remarks"></a>Remarques

Doit être 0, sauf en cas de non-lieu par d’autres protocoles qui étendent le protocole d’objet de message et de pièce jointe, comme indiqué dans [[MS-OXCMSG]](https://msdn.microsoft.com/library/daa9120f-f325-4afb-a738-28f91049ab3c%28Office.15%29.aspx).
