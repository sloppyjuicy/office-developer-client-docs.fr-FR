# <a name="pidtagroamingxmlstream-canonical-property"></a>Propriété canonique PidTagRoamingXmlStream

  
  
**S’applique à** : Outlook 2013 | Outlook 2016 
  
Contient un flux XML arbitraire.
  
|Propriété |Valeur |
|:-----|:-----|
|Propriétés associées :  <br/> |PR_ROAMING_XMLSTREAM  <br/> |
|Identificateur :  <br/> |0x7C08  <br/> |
|Type de données :  <br/> |PT_BINARY  <br/> |
|Domaine :  <br/> |Configuration  <br/> |
   
## <a name="remarks"></a>Remarques

Cette propriété contient un flux arbitraire de données XML. D’autres propriétés du message peuvent impliquer des schémas spécifiques à utiliser dans cette propriété.
