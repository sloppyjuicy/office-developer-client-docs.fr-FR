# <a name="pidtagsearchfoldertag-canonical-property"></a>Propriété canonique PidTagSearchFolderTag

  
  
**S’applique à** : Outlook 2013 | Outlook 2016 
  
Contient la valeur utilisée pour synchroniser ce message de définition avec le conteneur de dossier de recherche correspondant.
  
|Propriété |Valeur |
|:-----|:-----|
|Propriétés associées :  <br/> |PR_WB_SF_TAG  <br/> |
|Identificateur :  <br/> |0x6847  <br/> |
|Type de données :  <br/> |PT_LONG  <br/> |
|Domaine :  <br/> |Recherche  <br/> |
   
## <a name="remarks"></a>Remarques

Cette propriété est modifiée lorsque le message de définition est modifié. Elle doit modifier chaque itération, mais elle peut ne pas être unique.
