# <a name="pidnamecontentclass-canonical-property"></a>Propriété canonique PidNameContentClass

  
  
**S’applique à** : Outlook 2013 | Outlook 2016 
  
Contient une valeur de champ d’en-tête [RFC3282] Content-Class.
  
|Propriété |Valeur |
|:-----|:-----|
|Noms conviviaux :  <br/> |Aucun  <br/> |
|Jeu de propriétés :  <br/> |PS_INTERNET_HEADERS  <br/> |
|Nom de la propriété :  <br/> |Content-Class  <br/> |
|Type de données :  <br/> |PT_UNICODE  <br/> |
|Domaine :  <br/> |E-mail  <br/> |
   
## <a name="remarks"></a>Remarques

Pour définir la valeur de cette propriété, les clients MIME (Multipurpose Internet Message Extensions) doivent écrire un champ d’en-tête Content-Class avec la valeur souhaitée. Les lecteurs MIME doivent copier la valeur d’un champ d’en-tête Content-Class vers la valeur de cette propriété.
