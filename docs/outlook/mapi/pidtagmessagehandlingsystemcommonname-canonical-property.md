# <a name="pidtagmessagehandlingsystemcommonname-canonical-property"></a>Propriété canonique PidTagMessageHandlingSystemCommonName

  
  
**S’applique à** : Outlook 2013 | Outlook 2016 
  
Contient le nom commun du système de gestion des messages.
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |PR_MHS_COMMON_NAME, PR_MHS_COMMON_NAME_A, PR_MHS_COMMON_NAME_W  <br/> |
|Identificateur :  <br/> |0x3A0F  <br/> |
|Type de données :  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Domaine :  <br/> |Adresse  <br/> |
   
## <a name="remarks"></a>Remarques

Ces propriétés contiennent le nom commun du système de gestion des messages.
