# <a name="pidlidcurrentversion-canonical-property"></a>Propriété canonique PidLidCurrentVersion

  
  
**S’applique à** : Outlook 2013 | Outlook 2016 
  
Contient un type long qui représente le numéro de build de l’application Outlook pour un élément.
  
|Propriété|Valeur|
|:-----|:-----|
|Propriétés associées :  <br/> |dispidCurrentVersion  <br/> |
|Jeu de propriétés :  <br/> |PSETID_Common  <br/> |
|ID long (LID) :  <br/> |0x00008552  <br/> |
|Type de données :  <br/> |PT_LONG  <br/> |
|Domaine :  <br/> |Messagerie générale  <br/> |
