# <a name="pidnamebytecount-canonical-property"></a>Propriété canonique PidNameByteCount

  
  
**S’applique à** : Outlook 2013 | Outlook 2016 
  
Spécifie la taille, en octets, du fichier attaché au document.
  
|Propriété |Valeur |
|:-----|:-----|
|Noms conviviaux :  <br/> |Aucun  <br/> |
|Jeu de propriétés :  <br/> |PS_PUBLIC_STRINGS  <br/> |
|Nom de la propriété :  <br/> |Bytecount  <br/> |
|Type de données :  <br/> |PT_LONG  <br/> |
|Domaine :  <br/> |Courant  <br/> |
