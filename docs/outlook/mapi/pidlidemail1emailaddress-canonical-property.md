# <a name="pidlidemail1emailaddress-canonical-property"></a>Propriété canonique PidLidEmail1EmailAddress

  
  
**S’applique à** : Outlook 2013 | Outlook 2016 
  
Spécifie la première adresse e-mail du contact.
  
|Propriété |Valeur |
|:-----|:-----|
|Propriétés associées :  <br/> |dispidEmail1EmailAddress  <br/> |
|Jeu de propriétés :  <br/> |PSETID_Address  <br/> |
|ID long (LID) :  <br/> |0x00008083  <br/> |
|Type de données :  <br/> |PT_UNICODE  <br/> |
|Domaine :  <br/> |Contact  <br/> |
   
## <a name="remarks"></a>Remarques

La valeur de cette propriété doit être appropriée pour le type d’adresse spécifié pour cette adresse e-mail.
