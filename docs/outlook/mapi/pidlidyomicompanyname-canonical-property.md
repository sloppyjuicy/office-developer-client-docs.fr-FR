# <a name="pidlidyomicompanyname-canonical-property"></a>Propriété canonique PidLidYomiCompanyName

  
  
**S’applique à** : Outlook 2013 | Outlook 2016 
  
Spécifie la prononciation phonétique du nom d’entreprise du contact.
  
|Propriété|Valeur|
|:-----|:-----|
|Propriétés associées :  <br/> |dispidYomiCompanyName  <br/> |
|Jeu de propriétés :  <br/> |PSETID_Address  <br/> |
|ID long (LID) :  <br/> |0x0000802E  <br/> |
|Type de données :  <br/> |PT_UNICODE  <br/> |
|Domaine :  <br/> |Contact  <br/> |
