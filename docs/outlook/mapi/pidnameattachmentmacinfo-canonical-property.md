# <a name="pidnameattachmentmacinfo-canonical-property"></a>Propriété canonique PidNameAttachmentMacInfo

  
  
**S’applique à** : Outlook 2013 | Outlook 2016 
  
Contient une valeur de pièce jointe [RFC3282] qui est composée des données d’en-tête et de duplication de ressources appropriées utilisées par les clients de messagerie basés sur Macintosh.
  
|Propriété|Valeur|
|:-----|:-----|
|Noms conviviaux :  <br/> |Aucun  <br/> |
|Jeu de propriétés :  <br/> |PSETID_Attachment  <br/> |
|Nom de la propriété :  <br/> |AttachmentMacInfo  <br/> |
|Type de données :  <br/> |PT_BINARY  <br/> |
|Domaine :  <br/> |Pièce jointe de message  <br/> |
   
## <a name="remarks"></a>Remarques

Pour plus d’informations, consultez la section MS-OXCMAIL 2.2.4.2 Formats de fichier Apple.
