# <a name="pidlidemail1addresstype-canonical-property"></a>Propri t canonique PidLidEmail1AddressType

  
  
**S’applique à** : Outlook 2013 | Outlook 2016 
  
Spécifie le type d’adresse de la première adresse de messagerie.
  
|Propriété|Valeur|
|:-----|:-----|
|Propriétés associées :  <br/> |dispidEmail1AddrType  <br/> |
|Jeu de propriétés :  <br/> |PSETID_Address  <br/> |
|ID long (LID) :  <br/> |0x00008082  <br/> |
|Type de données :  <br/> |PT_UNICODE  <br/> |
|Domaine :  <br/> |Contact  <br/> |
   
## <a name="remarks"></a>Remarques

S’il est présent, cette valeur de propriété doit être un type d’adresse valide.
