# <a name="pidlidfax2addresstype-canonical-property"></a>Propriété canonique PidLidFax2AddressType

  
  
**S’applique à** : Outlook 2013 | Outlook 2016 
  
Spécifie le type d’adresse de l’adresse de télécopie du contact.
  
|Propriété |Valeur |
|:-----|:-----|
|Propriétés associées :  <br/> |dispidFax2AddrType  <br/> |
|Jeu de propriétés :  <br/> |PSETID_Address  <br/> |
|ID long (LID) :  <br/> |0x000080C2  <br/> |
|Type de données :  <br/> |PT_UNICODE  <br/> |
|Domaine :  <br/> |Contact  <br/> |
   
## <a name="remarks"></a>Remarques

Cette propriété, si elle est présente, doit être définie sur « FAX ».
