# <a name="pidlidcontactuserfield2-canonical-property"></a>PidLidContactUserField2 Canonical, propriété

  
  
**S’applique à** : Outlook 2013 | Outlook 2016 
  
Spécifie le deuxième champ sur le contact qui est destiné à une utilisation diverses pour le contact.
  
|Propriété |Valeur |
|:-----|:-----|
|Propriétés associées :  <br/> |dispidContactUserField2  <br/> |
|Jeu de propriétés :  <br/> |PSETID_Address  <br/> |
|ID long (LID) :  <br/> |0x00008050  <br/> |
|Type de données :  <br/> |PT_UNICODE  <br/> |
|Domaine :  <br/> |Contact  <br/> |
   
## <a name="remarks"></a>Remarques

Cette propriété contient du texte qui n’est pas lié à toute autre propriété spécifique au contact.
