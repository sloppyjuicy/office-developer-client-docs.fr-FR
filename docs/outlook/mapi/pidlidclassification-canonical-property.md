# <a name="pidlidclassification-canonical-property"></a>Propriété canonique PidLidClassification

  
  
**S’applique à** : Outlook 2013 | Outlook 2016 
  
Indique le nom de la classification des messages.
  
|Propriété |Valeur |
|:-----|:-----|
|Propriétés associées :  <br/> |dispidClassification  <br/> |
|Jeu de propriétés :  <br/> |PSETID_Common  <br/> |
|ID long (LID) :  <br/> |0x000085B6  <br/> |
|Type de données :  <br/> |PT_UNICODE  <br/> |
|Domaine :  <br/> |Messagerie générale  <br/> |
