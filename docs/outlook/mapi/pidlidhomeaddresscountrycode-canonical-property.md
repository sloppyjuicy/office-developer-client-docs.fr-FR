# <a name="pidlidhomeaddresscountrycode-canonical-property"></a>Propriété canonique PidLidHomeAddressCountryCode

  
  
**S’applique à** : Outlook 2013 | Outlook 2016 
  
Spécifie la partie du code du pays de l’adresse d’accueil du contact.
  
|Propriété|Valeur|
|:-----|:-----|
|Propriétés associées :  <br/> |dispidHomeAddressCountryCode  <br/> |
|Jeu de propriétés :  <br/> |PSETID_Address  <br/> |
|ID long (LID) :  <br/> |0x000080DA  <br/> |
|Type de données :  <br/> |PT_UNICODE  <br/> |
|Domaine :  <br/> |Contact  <br/> |
