|:-----|:-----|
|Propriétés associées :  <br/> |PR_AUTOCFG_USER_PASSWORD_W  <br/> |
|Identificateur :  <br/> |0x67FC  <br/> |
|Type de données :  <br/> |PT_UNICODE  <br/> |
|Domaine :  <br/> |Pièce jointe de message  <br/> |
