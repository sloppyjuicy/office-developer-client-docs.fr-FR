# <a name="pidtagbirthday-canonical-property"></a>Propriété canonique PidTagBirthday

  
  
**S’applique à** : Outlook 2013 | Outlook 2016 
  
Contient l’anniversaire du contact.
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |PR_BIRTHDAY  <br/> |
|Identificateur :  <br/> |0x3A42  <br/> |
|Type de données :  <br/> |PT_SYSTIME  <br/> |
|Domaine :  <br/> |Contact  <br/> |
   
## <a name="remarks"></a>Remarques

Cette propriété est l’une des propriétés qui fournit des informations d’identification et d’accès sur un contact. Ces propriétés sont définies par l’utilisateur et l’organisation de l’utilisateur.
