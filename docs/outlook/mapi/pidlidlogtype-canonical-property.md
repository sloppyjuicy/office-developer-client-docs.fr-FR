# <a name="pidlidlogtype-canonical-property"></a>Propriété canonique PidLidLogType

  
  
**S’applique à** : Outlook 2013 | Outlook 2016 
  
Contient le nom complet de l’application de journalisation (par exemple, « MSWord ») et est généralement un attribut de forme libre d’un message journal, généralement une chaîne.
  
|Propriété|Valeur|
|:-----|:-----|
|Propriétés associées :  <br/> |dispidLogType  <br/> |
|Jeu de propriétés :  <br/> |PSETID_Log  <br/> |
|ID long (LID) :  <br/> |0x00008700  <br/> |
|Type de données :  <br/> |PT_UNICODE  <br/> |
|Domaine :  <br/> |Journal  <br/> |
