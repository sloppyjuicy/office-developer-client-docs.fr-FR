# <a name="pidlidcontactuserfield3-canonical-property"></a>Propriété canonique PidLidContactUserField3

  
  
**S’applique à** : Outlook 2013 | Outlook 2016 
  
Spécifie le troisième champ sur le contact qui est destiné à une utilisation diverses pour le contact.
  
|Propriété|Valeur|
|:-----|:-----|
|Propriétés associées :  <br/> |dispidContactUserField3  <br/> |
|Jeu de propriétés :  <br/> |PSETID_Address  <br/> |
|ID long (LID) :  <br/> |0x00008051  <br/> |
|Type de données :  <br/> |PT_UNICODE  <br/> |
|Domaine :  <br/> |Contact  <br/> |
   
## <a name="remarks"></a>Remarques

Cette propriété contient du texte qui n’est pas lié à toute autre propriété spécifique au contact.
