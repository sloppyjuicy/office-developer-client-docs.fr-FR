# <a name="pidlidfreebusylocation-canonical-property"></a>Propriété canonique PidLidFreeBusyLocation

  
  
**S’applique à** : Outlook 2013 | Outlook 2016 
  
Spécifie un chemin d’URL à partir duquel un client peut récupérer les informations de disponibilité du contact sous la forme d’un fichier iCal, comme spécifié dans [[MS-OXCICAL]](https://msdn.microsoft.com/library/a685a040-5b69-4c84-b084-795113fb4012%28Office.15%29.aspx).
  
|Propriété |Valeur |
|:-----|:-----|
|Propriétés associées :  <br/> |dispidFreeBusyLocation  <br/> |
|Jeu de propriétés :  <br/> |PSETID_Address  <br/> |
|ID long (LID) :  <br/> |0x000080D8  <br/> |
|Type de données :  <br/> |PT_UNICODE  <br/> |
|Domaine :  <br/> |Contact  <br/> |
