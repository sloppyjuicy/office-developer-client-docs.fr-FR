# <a name="pidtaginitials-canonical-property"></a>Propriété canonique PidTagInitials

  
  
**S’applique à** : Outlook 2013 | Outlook 2016 
  
Contient les initiales des parties du nom complet du destinataire. 
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |PR_INITIALS, PR_INITIALS_A, PR_INITIALS_W  <br/> |
|Identificateur :  <br/> |0x3A0A  <br/> |
|Type de données :  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Domaine :  <br/> |Adresse  <br/> |
   
## <a name="remarks"></a>Remarques

Ces propriétés fournissent des informations d’identification et d’accès pour un destinataire. Elles sont définies par le destinataire et leur organisation.
