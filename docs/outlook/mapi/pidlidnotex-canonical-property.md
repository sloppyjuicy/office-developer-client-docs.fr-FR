# <a name="pidlidnotex-canonical-property"></a>Propri t canonique PidLidNoteX

  
  
**S’applique à** : Outlook 2013 | Outlook 2016 
  
Spécifie la distance, en pixels, entre le bord gauche de l’écran où une interface utilisateur affiche une note.
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |dispidNoteX  <br/> |
|Jeu de propriétés :  <br/> |PSETID_Note  <br/> |
|ID long (LID) :  <br/> |0x00008B04  <br/> |
|Type de données :  <br/> |PT_LONG  <br/> |
|Domaine :  <br/> |Sticky Note  <br/> |
