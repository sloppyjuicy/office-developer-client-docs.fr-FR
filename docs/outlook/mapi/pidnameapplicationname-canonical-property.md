# <a name="pidnameapplicationname-canonical-property"></a>Propriété canonique PidNameApplicationName

  
  
**S’applique à** : Outlook 2013 | Outlook 2016 
  
Spécifie l’application qui peut ouvrir le fichier attaché au document.
  
|Propriété |Valeur |
|:-----|:-----|
|Noms conviviaux :  <br/> |Aucun  <br/> |
|Jeu de propriétés :  <br/> |PS_PUBLIC_STRINGS  <br/> |
|Nom de la propriété :  <br/> |AppName  <br/> |
|Type de données :  <br/> |PT_UNICODE  <br/> |
|Domaine :  <br/> |Courant  <br/> |
