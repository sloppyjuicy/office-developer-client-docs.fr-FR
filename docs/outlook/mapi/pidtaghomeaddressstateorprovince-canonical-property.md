# <a name="pidtaghomeaddressstateorprovince-canonical-property"></a>Propriété canonique PidTagHomeAddressStateOrProvince

  
  
**S’applique à** : Outlook 2013 | Outlook 2016 
  
Contient la partie État ou province de l’adresse d’un utilisateur.
  
|Propriété|Valeur|
|:-----|:-----|
|Propriétés associées :  <br/> |PR_HOME_ADDRESS_STATE_OR_PROVINCE, PR_HOME_ADDRESS_STATE_OR_PROVINCE_A, PR_HOME_ADDRESS_STATE_OR_PROVINCE_W  <br/> |
|Identificateur :  <br/> |0x3A5C  <br/> |
|Type de données :  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Domaine :  <br/> |Utilisateur de messagerie MAPI  <br/> |
   
## <a name="remarks"></a>Remarques

Ces propriétés sont définies par l’utilisateur ou l’organisation de l’utilisateur.
