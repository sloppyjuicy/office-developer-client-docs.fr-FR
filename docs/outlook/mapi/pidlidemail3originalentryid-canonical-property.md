# <a name="pidlidemail3originalentryid-canonical-property"></a>Propriété canonique PidLidEmail3OriginalEntryId

  
  
**S’applique à**: Outlook 
  
Spécifie **propriété EntryId** de l’objet qui correspond à l’adresse de messagerie tiers. 
  
|||
|:-----|:-----|
|Propriétés associées :  <br/> |dispidEmail3OriginalEntryID  <br/> |
|Jeu de propriétés :  <br/> |PSETID_Address  <br/> |
|ID de type long (capot) :  <br/> |0x000080A5  <br/> |
|Type de données :  <br/> |PT_BINARY  <br/> |
|Zone :  <br/> |Contact  <br/> |
   
## <a name="remarks"></a>Remarques

La valeur de cette propriété doit être un uniques **EntryId** pour cette adresse électronique ou un objet de carnet d’adresses **EntryId**valide.
