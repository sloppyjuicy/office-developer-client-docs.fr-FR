# <a name="pidtaghomeaddressstreet-canonical-property"></a>PidTagHomeAddressStreet Canonical, propriété

  
  
**S’applique à** : Outlook 2013 | Outlook 2016 
  
Contient la partie rue de l’adresse d’un utilisateur.
  
|Propriété|Valeur|
|:-----|:-----|
|Propriétés associées :  <br/> |PR_HOME_ADDRESS_STREET, PR_HOME_ADDRESS_STREET_A, PR_HOME_ADDRESS_STREET_W  <br/> |
|Identificateur :  <br/> |0x3A5D  <br/> |
|Type de données :  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Domaine :  <br/> |Utilisateur de messagerie MAPI  <br/> |
   
## <a name="remarks"></a>Remarques

Ces propriétés sont définies par l’utilisateur ou l’organisation de l’utilisateur.
