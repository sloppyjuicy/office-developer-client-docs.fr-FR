# <a name="len-function-visioshapesheet"></a>LEN Function (VisioShapeSheet)

Renvoie le nombre de caractères dans une chaîne de texte.
  
## <a name="syntax"></a>Syntaxe

LEN (***texte*** )
  
### <a name="parameters"></a>Paramètres

|**Nom**|**Requis/Facultatif**|**Type de données**|**Description**|
|:-----|:-----|:-----|:-----|
| *text* <br/> |Requis  <br/> |**String** <br/> |Chaîne de caractères dont vous souhaitez obtenir la longueur. |

### <a name="return-value"></a>Valeur renvoyée

Numérique
  
## <a name="example"></a>Exemple

LEN ("75014 Paris")
  
Renvoie 11.
