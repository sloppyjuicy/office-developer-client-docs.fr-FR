# <a name="type-function"></a>Fonction TYPE

Renvoie le type interne d’un objet sous forme de nombre. 
  
## <a name="syntax"></a>Syntaxe

TYPE ()
  
### <a name="return-value"></a>Valeur renvoyée

Nombre
