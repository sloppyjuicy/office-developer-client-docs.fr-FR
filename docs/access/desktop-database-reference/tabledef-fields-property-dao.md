# <a name="tabledeffields-property-dao"></a>TableDef.Fields Property (DAO)


**S’applique à**: Access 2013, Office 2013

Renvoie une collection **Fields** qui représente tous les objets **Field** stockés pour l'objet spécifié. En lecture seule.

## <a name="syntax"></a>Syntaxe

*expression* . Champs

*expression* Variable qui représente un objet **TableDef** .
