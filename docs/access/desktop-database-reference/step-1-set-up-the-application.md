# <a name="step-1-set-up-the-application"></a>Étape 1 : configurer l'application


**S’applique à**: Access 2013 | Office 2013

## <a name="step-1-set-up-the-application"></a>Étape 1 : configurer l'application

Créez un répertoire virtuel IIS nommé « XMLPersist » avec des autorisations de script. Créez deux nouveaux fichiers texte dans le dossier vers lequel pointe le répertoire virtuel : l'un nommé « XMLResponse.asp » et l'autre « Default.htm ».
